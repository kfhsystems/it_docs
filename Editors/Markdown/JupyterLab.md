## Installing a Theme
```bash
# Themes:
python3 -m pip install jupyterthemes
python3 -m pip install theme-darcula
python3 -m pip install catppuccin-jupyterlab

# To see available theme:
jt -l 
# Output:
Available Themes:
   chesterish
   grade3
   gruvboxd
   gruvboxl
   monokai
   oceans16
   onedork
   solarizedd
   solarizedl
   
```
