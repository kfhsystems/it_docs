# Vim FAQ
## How do I select multiple lines and the comment them out?
1. In the ***Command Mode*** , type **V** ( capital “**v**”) and start selecting
2. After selecting the lines you have to press  **Shift** + **:** 
3. You will see **:'<,'>**, continue typing : **s/^/#** **/** ( there is a space before last **/**)
4. **Enter**
